# Teacher Management System

A full-stack Teacher Management System developed using MERN Stack.

## Features

### Admin

- Admin Login
- Add Teacher
- View Teachers
- Edit Teacher Information
- Delete Teacher
- Manage Teacher Records

### Teacher

- Teacher Login
- View Personal Information
- View Schedule
- Secure Dashboard Access

## Technologies Used

### Frontend
- React.js
- React Router DOM
- CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### Authentication
- JWT Authentication
- Bcrypt Password Hashing

### Additional Features
- Email Notifications using Nodemailer
- Route Protection
- Role-Based Access Control

## Project Structure

Frontend:


Backend:
```

├── config/

```

## Installation

Frontend:

```bash
npm install
npm run dev
```

Backend:

```bash
npm install
npm start
```
