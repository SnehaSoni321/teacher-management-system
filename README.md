# Teacher Management System

A full-stack Teacher Management System built using React, Node.js, Express.js and MongoDB.

## Features
- User Signup
- User Login
- Password Hashing using bcrypt
- JWT Authentication (1 hour expiry)
- Admin Dashboard
- Add Teacher
- View Teachers
- Delete Teacher
- Notification Bell
- Logout

## Tech Stack
- HTML
- CSS
- JavaScript
- React.js
- Node.js
- Express.js
- MongoDB

## Project Structure
teacher-management-system
- frontend
- backend

## Run Project

### Frontend
```bash
npm run dev
```

### Backend
```bash
npm start
```
